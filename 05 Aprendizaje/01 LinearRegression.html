Regresión lineal¶

La regresin lineal es un mtodo de aprendizaje supervisado, donde dado un elemento de entrada $x \in \mathbb{R}^d$ se busca aprender una funcin $f(x; w) \in \mathbb{R}$ que estime valores reales para cada entrada $x$. Como su nombre lo dice, esta estimacin la hace de manera lineal. Es decir, estima una recta, una funcin lineal. De tal forma que $f: X \to Y$ es lineal. Si los datos tienen una correlacin lineal alta, la regresin lineal ser un buen aproximador de estos valores. Podemos ver tambin al valor $f(x;w)$ como el valor esperado para $x$.

A continuacin presentamos un ejemplo simple de regresin lineal con datos de la paquetera sklearn que describen casas y sus precios. El objetivo es estimar la funcin que dada las caractersticas de una casa regrese un precio estimado.

Estadísticas y visualización de los datos¶

Una parte importante del aprendizaje automtico es su relacin con el anlisis estadstico. Por tanto, antes de pasar a aplicar el algoritmo para estimacin de la regresin lineal podemos computar algunas estadsticas. Una de la ms importantes en este caso, en tanto modelo lineal, es la correlacin lineal. Con la tabla de pandas que hemos creado, podemos definir una correlacin entre las variables. Ya que buscamos establecer una regresin lineal, los elementos que muestren una mayor correlacin con la variable $Y$ (precio) sern los que nos servirn mejor.

La correlacin de Pearson (lineal) se estima de la siguiente forma:

$$\rho(X,Y) = \frac{Cov(X,Y)}{\sigma_X \sigma_Y}$$

Donde $Cov(X,Y)$ es la covarianza entre las variables $X$ e $Y$, la cul se clcula como $Cov(X,Y) = \mathbb{E}\big[ (X-\mu_x)(Y-\mu_y) \big]$ donde $\mu_i$ son las medias de las variables correspondientes. Asimismo, $\sigma_i$ es la desviacin estandar de las variables.

Regresión lineal¶

La regresin lineal, como ya lo sealamos, es un modelo de aprendizaje supervisado lineal que estima una funcin lineal $f: \mathbb{R}^d \to \mathbb{R}$ debe ajustar los datos de entrenamiento, de tal forma que pueda predecir datos nuevo. El modelo de regresin lineal se divide en los siguientes mdulos:

  1. Prediccin: El modelo de regresin lineal estima un valor real a partir de un elemento de entrada $x$ a partir de la funcin lineal: $$f(x) = \sum_{i=1}^d w_i x_i + b= wx+b$$ De tal forma que podemos ver los parmetros a aprender como un vector $w \in \mathbb{R}^d$. El valor $b$ conocido como bias o sesgo es un valor de traslacin para que la recta o hiperplano que ajusta los datos no siempre sea centrada en el 0.
  2. Aprendizaje: El aprendizaje se determina como un problema de optimizacin. En tanto que lo que buscamos es que $f(x)$ sea lo ms cercano a $y$, la funcin de riesgo/objetivo que definimos es un error cuadrtico: $$R(w) = \frac{1}{2} ||f(X) - Y||^2$$ El mnimo de esta funcin se puede estimar obteniendo la derivada de la funcin e igualando a cero. La derivada de la funcin est dada por:
\begin{align} \nabla_w R(w) &= (f(X) - Y)X \\ &= (Xw - Y)X \\ &= X^TXw-X^TY \end{align}

De aqu que, cuando $\nabla_w R(w) = 0$, entonces: $$w = (X^T X)^{-1}X^TY$$ \textbf{Nota}: Para simplificar el mtodo, agreamos una columna con 1's a los datos de entrada, de tal forma que $x' = [x;1]$. este 1 servir para representar el bias, de tal forma que la funcin $f(x) = wx'$; es decir, el bias ser una entrada (la ltima) del vector w.

La clase que definimos incorpora estas funciones: prediccin (predict) y aprendizaje (fit).

In [8]:
class LinearRegression(object):
    """Clase del modelo de regresin lineal"""
    def __init__(self):
        #Parmetros
        w = None
        b = None
        
    def predict(self, x):
        """Prediccin del modelo"""
        f = np.dot(x,self.w) + self.b
        
        return f
        
    def fit(self, x,y):
        """Estimacin de los parmetros"""
        #Se concatena una columna de 1's a X
        Xb = np.concatenate((x, np.ones((len(x),1))), axis=1)
        #XX^-1 
        term1 = np.linalg.inv(np.dot(Xb.T,Xb))
        #XY
        term2 = np.dot(Xb.T,y)
        #Vector solucin
        w = np.dot(term1,term2)
        #Guarda parmetros
        self.w = w[:-1]
        self.b = w[-1]

Ahora bien, aplicamos el modelo a nuestro problema y lo entrenamos con los datos de entrenamiento para obtener los valores de los parmetros.

Evaluación de los datos¶

Finalmente, podemos ver cmo se comporta el error a travs del dataset de entrenamiento y de evaluacin. El error que nos interesa reportar (para determinar la capacidad de generalizacin de nuestro modelo) es el error de evaluacin. Ya que estamos usando un modelo de regresin, podemos estimar el error cuadrtico y el score $R^2$.

El error cuadrtico medio se estima como:

$$MSE = \frac{1}{N}\sum_{i=1}^N (y_i - \hat{y}_i)^2$$

Donde $y_i$ son los valores de regresin reales, mientras que $\hat{y}_i$ son los valores predichos por el modelo. Por otra parte, el score $R^2$ se estima com:

$$R^2 = 1 - \frac{\sum_{i=1}^N (y_i - \hat{y}_i)^2}{\sum_{i=1}^N (y_i - \hat{y}_i)}$$

Recordemos que las mtricas de evaluacin se hacen sobre los datos de evaluacin, no los de entrenamiento. Para evaluar, utilizamos las metricas disponibles en skleanr.

Pero la regresin lineal puede estimarse tomando en cuenta $d$ varibales de entrada. As, podemos tomar las 13 variables de los datos y ver cules son las que muestran mayor influencia en la decisin del precio. En este caso, nuestra funcin de regresin es una funcin $f:\mathbb{R}^d \to \mathbb{R}$.

En realidad, no hay mayores cambios al modelo, pues este puede trabajar con entradas de diferentes dimensiones. Slo que ahora los parmetros aprendidos sern 13 (o en general $d$) en lugar de slo 1 (sin considerar el sesgo $b$). El procedimiento es el mismo.
