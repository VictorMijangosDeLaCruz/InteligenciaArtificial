Regresión logística¶

El modelo de regresin logstica es un modelo grfico no dirigido que estima una probabilidad de una distribucin binaria $X \sim Ber(p)$ donde $p =p(y=1|x)$ es la probabilidad de que el observable $x$ sea parte de la clase y se estima por medio de la funcin logsitica:

$$p(y=1|x) = \frac{1}{1+exp\{-(\theta x + \theta_0\})\}}$$

A partir de aqu se puede predecir una clase que clasifique al vector de entrada. Presentamos una implementacin sencilla que utiliza el algoritmo de gradiente descendiente para aprender los pesos.

Aprendizaje en el modelo de regresión logística¶

Para realizar el aprendizaje en el modelo de regresin logstica primero que nada generaremos el modelo, como sabemos el modelo aplica la funcin logstica:

$$f(x) = \frac{1}{1 + e^{-\theta x - \theta_0}}$$

Donde $\theta$ y $\theta_0$ definen los factores del modelo. El objetivo es entonces estimar esto parmetros para que la prediccin hecha por el modelo sea correcto.

Función logística¶

Definimos, en principio, la funcin logstica.

In [5]:
#Funcin logstica
logist = lambda a: 1./(1.+np.exp(-a))

La funcin logstica toma valores entre 0 y 1 y puede, por tanto, interpretarse como una probabilidad. En este caso es la probabilidad de que los datos observados pertenezcan a la clase 0. El comportamiento de la funcin logstica es como de S, como puede observarse a continuacin.

Estimación de los parámetros¶

Para obtener los valores $\theta_i$, $i=\{0,1,...,d\}$ adecuados, usaremos el algoritmo del gradiente descendiente. Este es un algoritmo sencillo que depende del gradiente de una funcin de costo o riesgo que depende de los parmetros, $R(\theta)$.

El mtodo del gradiente descendiente actualiza los parmetros en base al gradiente y a un factor $\eta$ conocido como la tasa de aprendizaje que determina que tanto cambia cada parmetro con respecto al gradiente:

$$\theta_i \leftarrow \theta_i - \eta \nabla_iR(\theta)$$

En el caso de la regresin logstica para problemas binarios, nuestra funcin de costo estar dada por:

$$R(\theta) = -\Big( y \log p(y=1|x) + (1-y)\log \big( 1- p(y=0|x) \big) \Big)$$

En este caso, la probabilidad se obtiene con la funcin logstica. Notamos que las derivadas parciales (que definen el gradiente) son:

$$\frac{\partial R(\theta)}{\partial \theta_i} (p(y=1|x) - y) x_i$$

Y en el caso de $\theta_0$ se puede denotar que $x_0= 1$ siempre. Por tanto, la actualizacin de los parmetros se realzar como:

$$\theta_i \leftarrow \theta_i - \eta \big( f(x) - y \big) \cdot x_i$$

Donde $f(x) = p(y=1|x)$ es la funcin logstica. Para actualizar los parmetros utilizaremos una tasa de aprendizaje $\eta$ e iteraremos varias veces el procedimiento.

Finalmente, una vez obtenido los pesos, podemos estimar las probabilidades de los objetos que estamos trabajando. En este caso, se estima la probabilidad para la calse 1 como $f(x)$ y para la clase 0 obtenemos que se puede sacar de la anterior como $1-f(x)$.

Asimismo, podemos estimar cul es la clase que mejor se ajusta al dato observable simplemente viendo su probabilidad; si esta supera el 0.5 ser de la clase 1, sino pertenecer a la clase 0.

$$\hat{y} = \begin{cases} 1 & \text{si } f(x) > 0.5 \\ 0 & \text{en otro cas} \end{cases}$$

class LogisticRegression():
    def __init__(self, lr=0.1):
        #Rango de aprendizaje
        self.lr = lr
        #Iniciamos theta_i con 0s
        self.theta = None #np.zeros(d)
        #Iniciamos theta_0 = 0
        self.theta0 = 0
        
    def fit(self, x,y, max_its=10):
        #Tamao de los datos
        m,d = x.shape
        #Incializa theta
        self.theta = np.random.rand(d)/np.sqrt(d)
        
        stop = False
        t = 0
        while  stop == False:
            #Revisamos cada uno de los ejemplos
            for x_i, y_i in zip(x,y):
                #Calculamos funcin logstica
                f = logist(np.dot(self.theta,x_i)+self.theta0)
                #Actualizamos los parmetros
                self.theta = self.theta - self.lr*(f-y_i)*x_i
                self.theta0 = self.theta0 - self.lr*(f-y_i)

            t += 1
            #Criterio de paro
            if t > max_its: 
                stop = True
                
    def predict_proba(self, x):
        #Calcula el valor lineal
        a = np.dot(x, self.theta)+self.theta0
        #Aplica la funcin logstica
        f = logist(a)
        
        return f
    

    def predict(self, x):
        #Obtiene probabilidades
        prob = self.predict_proba(x)
        #Obtiene las clases
        y_pred = 1*(prob > 0.5)
        
        return y_pred

Evaluación del modelo¶

El modelo de regresin logstica no obtiene una clase como salida, sino una probabilidad. Esta probabilidad est definida como:

$$p(Y=1|x) = \frac{1}{1+exp\{-(\theta x +\theta_0)\}}$$

Y determina la probabilidad que la clase sea 1. El modelo de regresin logstica asume que los datos de entrada pertenecen a clases binarias; es decir, asumen que la variable $Y$ toma slo los valores 0 y 1. Por tanto, es fcil obtener la probabilidad de la clase 0 como: $$p(Y=0|x) = 1-p(Y=1|x)$$ De esta forma, podemos obtener las probabilidades para ambas clases.

Asimismo, podemos elegir una clase segn la probabilidad. Claramente tenemos que la clases se puede determinar pro medio de la probabilidad como:

$$\hat{y} = \arg\max_y p(Y=y|x) = \begin{cases} 1 & \text{si } p(Y=1|x) > 0.5 \\ 0 & \text{en otro caso} \end{cases}$$

Los parmetros $\theta_i$ determinan los factores del modelo, pero tambin nos dan informacin sobre el estado de creecnias del modelo.
