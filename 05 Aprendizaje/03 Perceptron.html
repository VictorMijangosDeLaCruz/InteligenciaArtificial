En las redes neuronales, los pesos pueden interpretarse como la fuerza de las conexiones que unen dos neuronas entre s. De esta forma, los pesos de la red neuronal determinan cmo se comportar sta cuando reciba un vector de entrada: de los pesos depende la clase a la que pertenecer el ejemplo de entrada.

Sin duda, cuanto ms compleja sea la red, ms compleja ser la interpretacin de las interacciones de las neuronas a travs de los pesos. Aqu presentamos un ejemplo simple: el perceptrn.

Los datos pueden verse como vectores aleatorios; es decir, como vectores cuyas entradas son variables aleatorias. As, un vector aleatorio se ve como:

$$X^T = [X_1, X_2, ..., X_n]$$

Cuando muestreamos los ejemplos, esperamos que estos sean iid (independientes e idnticamente distribuidos); esto implica que cada vector ejemplo $X$ no depende de ningn otro ejemplo en el dataset. $$\hat{y} = \arg\max_y p(Y=y|X_1,X_2,...,X_n)$$

Aplicacin del algoritmo del perceptrn

El perceptrn es un algoritmo de aprendizaje estadstico que busca estimar las clases de un conjunto de datos a partir de la funcin de decisin:

$$f(x; \theta) = \begin{cases} 1 & \text{ si } wx + b > 0 \\ 0 & \text{ si } wx + b \leq 0\end{cases}$$

Aqu $x$ es un vector de entrada. La funcin f depende de $\theta = {w,b}$ que representa los parmetros o los pesos de la red: $w$ contienen los pesos de las conexiones y $b$ es el bias. Definimos esta funcin a continuacin:

In[5]:
#Funcin de decisin
f = lambda a: (a >= 0).astype(np.int64)

Algoritmo del perceptrn

El objetivo del algoritmo del perceptrn es obtener los parmetros $\theta$ adecuados para que la red pueda clasificar adecuadamente los ejemplos. Este algoritmo es iterativo y busca minimizar el cunto se equivoca la red en cada paso.

Antes de entrar de lleno al este algoritmo, agregaremos una columna ms a la matriz de datos de entrenamiento; esto con el objetivo de que esta columna represente el bias, de tal forma que slo actualicemos sobre un vector de parmetros:

Para inicializar el algoritmo del perceptron, tomaremos un vector $w \in \mathbb{R}^{n+1}$, donde $n$ es el nmero de variables (la dimensin de los vectores ejemplos) y el $+1$ responde al bias. Asimismo, estableceremos un nmero mximo de iteraciones (en caso de que el algoritmo no sea capaz de minimizar el error) y, finalmente, estableceremos un rango de aprendizaje.

Para entrenar la red, utilizaremos el algoritmo del perceptrn. Este consta de dos partes:

  • Forward. Para cada ejemplo $x$, se computar la funcin $f(x; \theta)$
  • Backward. Se actualizarn los pesos de $\theta = {w,b}$ por medio de la regla: $$w_i \leftarrow w_i - \eta (f(x;\theta)-y_x)x_i$$ Donde $y_x$ es la clase a la que pertenece el ejemplo $x$.

El algoritmo iterar hasta que el error $f(x; \theta) -y$ sea 0 o cuando se cumpla el nmero mximo de iteraciones.

In[6]:
class Perceptron():
    """Clase para modelo del perceptrn"""
    def __init__(self, lr=0.1):
        self.lr = lr
        self.w = None
        self.b = 1
        
    def fit(self, x,y, max_its=100):
        """Entrenamiento del modelo"""
        #Tamao de los datos
        m,d = x.shape
        #Inicializa theta
        self.w = np.random.rand(d)/np.sqrt(d)
        
        stop = False
        t = 0
        while  stop == False:
            #Revisamos cada uno de los ejemplos
            err_it = 0
            for x_i, y_i in zip(x,y):
                #Calculamos funcin logstica
                out = f(np.dot(self.w,x_i)+self.b)
                #Actualizamos los parmetros
                self.w = self.w - self.lr*(out-y_i)*x_i
                self.b = self.b - self.lr*(out-y_i)
                #Acumula error
                err_it += out-y_i

            t += 1
            #Criterio de paro
            if err_it == 0 or t > max_its: 
                stop = True
                
        #Guarda los parmetros aprendidos
        #self.w = w[:-1]
        #self.b = w[-1]
                
    def predict(self,x):
        """Prediccin de clases"""
        #Prediccin de la clase
        y_pred = f( np.dot(x, self.w) + self.b )
                
        return y_pred

Relacin del perceptrn con regresin logstica

Los parmetros aprendidos en el perceptrn y en la regresin logstica estn relacionados, pues como podemos ver, la regresin logstica se basa en la funcin logstica, que determina la probabilidad $p(Y=1|x)$ de la siguiente forma:

$$\sigma(a) = \frac{1}{1+e^{-a}}$$

La cual es 0.5 cuando $a=0$, pero si $a < 0$ entonces el valor de la funcin (probabilidad) ser menor a 0.5; por su parte si $a > 0$, entonces la funcin ser mayor a 0.5. En cualquier caso, podemos tomar la decisin con base en la funcin logstica como:

$$f_\sigma(x; \theta) = \begin{cases} 1 & \text{ si } \sigma(wx + b) > 0.5 \\ 0 & \text{ si } \sigma(wx + b)\leq 0.5\end{cases}$$

In[10]:
#Funcin sigmoide
sig = lambda x: 1./(1.+np.exp(-x))

Sin embargo, notamos que en el perceptrn nos saltamos la fase de calcular un valor de probabilidad y decide con base en si el valor $wx +b$ es mayor o menor a 0. Pero podemos concluir que los valores de los parmetros tanto para el perceptrn y la regresin logstica tomarn las mismas decisiones en la clasificacin. Por tanto, mediante el modelo del perceptrn, si bien modelo de clasificacin, tambin podemos estimar probabilidades de las clases.
