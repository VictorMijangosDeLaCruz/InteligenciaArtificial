k-Means¶

El algoritmo de k-means o k-medias es un mtodo de agrupamiento basado en las vecindades de los datos vistos como puntos en un espacio vectorial.

El mtodo busca estimar las medias de los clsters de tal forma que la varianza de estas medias sea mnima; de tal forma que los grupos o clsters obtenidos contengan puntos parecidos entre s, en donde la similitud se basa en la cercana bajo una mtrica de espacio vectorial.

El algoritmo de $k$-medias se basa en los siguientes pasos:

  1. Se generan centroides (medias) de manera aleatoria.
  2. Los puntos del dataset se agrupan en el clster, $\hat{c}$, cuya media es la ms cercana; esto es: $$\hat{c} = \arg\min_j ||x-\mu_j||^2$$
  3. Se recalculan las medias con base en los puntos que pertenencen a su clster. Esto es: $$\mu_j = \frac{1}{|\hat{c}|} \sum_{x \in \hat{c}} x$$
  4. Se repiten los pasos 2 y 3 hasta que no se hagan ms reasignaciones de las medias.

class kMeans():
    """
    Clase que define el mtodo de k-medias.
    """
    def __init__(self, n_clusters=2, init='random'):
        #Nmero de clsters
        self.n_clusters = n_clusters
        #Clsters vectores
        self.clusters = defaultdict(list)
        #Clsters con nombres
        self.named_clusters = defaultdict(list)
        #Centroides
        self.centroids = None
        #Inixializacin
        self.init = init
        
    def fit(self, X, names=None):
        """
        Aprende las medias/cendtroides adecuados dados los datos no-supervisados.
        """
        #Inicializacin de parmetros
        if self.init == 'data':
            C = X[np.random.choice(X.shape[1], self.n_clusters)]
        if self.init == 'random':
            C = np.random.uniform(np.min(X),np.max(X), size=(self.n_clusters, X.shape[1]))
        #Paro
        stop = False
        while stop == False:
            #Clculo de distancias con respecto a c
            dists = np.array([np.linalg.norm(X-c, axis=1) for c in C])
            #Cluster ms cercano
            clusts = np.argmin(dists,axis=0)
            #Clusterizacin de los datos
            if names != None:
                data = names
            else:
                data = [i for i in range(X.shape[0])]
            for k,v,n in zip(clusts,X,data):
                self.clusters[k].append(v)
                self.named_clusters[k].append(n)
                
            #Actualizacin de C
            new_c = np.array([sum(vecs)/len(vecs) for clas, vecs in self.clusters.items()])
            
            #Criterio de paro
            if C.all() == new_c.all():
                stop = True
                self.centroids = C
            else:
                #Actualizacin de parmetros
                C = new_c

Aplicación del algoritmo¶

Para probar el algoritmo generamos datos aleatorios en $\mathbb{R}^2$ en base a una distribucin normal. Estos datos perteneceran a dos clsters.

def plot_dist(mu):
    """
    Funcin para visualizar las distribuciones.
    """
    #Distribucin normal
    a = np.linspace(-3,3,500)
    b = np.linspace(-3,3,500)
    A, B = np.meshgrid(a,b)
    pos = np.empty(A.shape + (2,))
    pos[:, :, 0] = A; pos[:, :, 1] = B
    rv = multivariate_normal(mu) #, np.dot(desv.T,desv))

    #Ploteo
    fig = plt.figure()
    ax = fig.gca(projection='3d')
    ax.plot_surface(A, B, rv.pdf(pos),cmap='viridis',linewidth=0)

#Visualizacin de las distribuciones

Evaluación¶

La evaluacin en el agrupamiento, al ser no-supervisado, no es sencilla, pues se requerira de una especie de supervisin para poder comparar la calidad de los clsters.

En este caso usamos un gold estandard; es decir, un agrupamiento que considerariamos ptimo para los puntos. Este gold estandard est en base a la forma en que generamos los datos, pero en general depende de los resultados esperados.

Para evaluar utilizamos la pureza, definida como:

$$purity = \frac{1}{N} \sum_i \max_j |c_i \cap \hat{c}_j|$$
